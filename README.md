# 特性一览
## 视觉搜索
(icon)
按下设定的 “框索未知” 快捷键，呼出界面后，使用鼠标框中要搜索的图片即可调用以图搜图引擎搜索。支持 Google Lens, Yandex. 半支持百度、必应。

## 划词搜索
(icon)
在外部（浏览器、聊天等）选中文字后，按下设定的 “框索未知” 快捷键，会直接在框索未知窗口中搜索。支持大多主流搜索引擎。若选中文字为链接则会直接用默认浏览器打开链接。

## 文字识别
(icon)
按下设定的 “框索未知” 快捷键，呼出界面后，框选屏幕中的文字即可自动识别填入文字处理框并调用搜索引擎搜索文字。搜索完成也可以复制识别的文字。

## 翻译
(icon)
呼出 “框索未知” 界面，选中翻译模式后，框选文本即可识别并调用翻译。同时也可以直接翻译文字处理框中的文字。

## 扫码
(icon)
呼出 “框索未知” 界面后，会自动识别屏幕中的二维码并弹出卡片，左键点击卡片即可进入，右键卡片解析二维码源内容。

## 截图
(icon)
按下设定的截图快捷键即可快速截取屏幕。在框索未知窗口中也可以局部保存框中图片

## 取色
(icon)
呼出 “框索未知” 界面，选中取色工具后，点击屏幕中的像素点即可复制色号到剪切板。
